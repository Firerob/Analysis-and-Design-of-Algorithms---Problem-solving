## Solved Problems by Paradigm

### 1. Divide and Conquer
*   **Focus:** Solving problems by recursively breaking them down into independent subproblems.

### 2. Greedy Algorithms
*   **Focus:** Problems requiring local optimization to reach an optimal global solution.

### 3. Dynamic Programming (DP)
*   **Focus:** Multi-stage decision problems with overlapping subproblems, solved using efficient memoization or bottom-up tabulation.

### 4. Backtracking
*   **Focus:** Systematic state-space tree searches for constraint satisfaction problems.
